# 使用python发包实现江苏大学系统登陆（不是利用自动化，直接破破解滑块登陆、密码加密）此项目只是为了学习交流，请勿传播Thank you。

思路：江苏大学登录系统通过F12抓包可以看到账号没有加密，但是密码进行了加密，并且在这个过程当中是有滑块进行验证的一个过程。所以在这个过程当中我们只需要通过发包模拟加密和滑块验证的过程就可以实现登录的一个过程。
